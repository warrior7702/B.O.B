# Category: Web Scraping & Research Tools

## Scrapling
**What:** Undetectable web scraping library
**Features:**
- Bypasses Cloudflare Turnstile out of the box
- Impersonates real browser TLS fingerprints
- Full browser automation with stealth mode
- Built-in proxy rotation
- Concurrent crawling with pause/resume
- Adaptive mode: relearns page structure when sites redesign
- 784x faster than BeautifulSoup (benchmarked)
- MCP server for AI agent integration
- Interactive shell for terminal scraping

**Install:** `pip install scrapling`
**License:** BSD-3, 100% open source

## Vercel Agent Browser
**What:** CLI for AI browser automation
**Features:**
- Navigate any website
- Click, type, interact with elements
- Take screenshots & extract data
- Persist sessions, cookies & auth
- Run complete workflows autonomously

**Link:** https://github.com/vercel-labs/agent-browser

## Use Cases
- Automated data extraction
- Competitor monitoring
- Price tracking
- Content aggregation
- Form automation

## Security Note
Never connect scraping tools directly to sensitive systems (email, CRMs, public-facing systems). Risk of prompt injection attacks.
