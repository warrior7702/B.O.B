# Category: Development Tools & Utilities

## CodexBar
**What:** macOS menu bar app tracking AI usage limits in real-time
**Tracks:** Codex, Claude Code, Cursor, Gemini, Copilot, Kiro, etc.
**Features:**
- Lives in menu bar (always visible)
- Shows session + weekly usage per tool
- Countdown timer for limit resets
- Reads local data (no passwords, no cloud)
- Built-in CLI for terminal/CI checks

**Install:** `brew install --cask steipete/tap/codexbar`
**Stats:** 1.7K GitHub stars, 1,177 commits, 28 contributors
**License:** MIT, 100% open source
