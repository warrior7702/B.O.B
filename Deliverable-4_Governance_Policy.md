# Deliverable 4: Governance/Policy Draft (concise)

Opt-in model
- Per-session opt-in for data types (identity/config hints auto-allowed with consent flag; memory hints require explicit opt-in)

Approval flow
- Ticket-based approvals with digest previews; channel selection; timeout; escalation

Retention
- Identity/config hints: 30-90 days (config-dependent); memory hints: 0-90 days; restore logs: as rollback window

Compliance and audits
- Audit logs; consent logs; data minimization; per-item revocation
