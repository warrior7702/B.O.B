## Lessons Learned

### "I Can't Do That" Rules (New)
1. Can't browse Amazon links directly → Ask for product name/model
2. Can't resolve DNS when host offline → Report clearly, don't guess
3. Can't access hardware without credentials → Ask, don't attempt bypass
4. Can't fix hardware limitations → Suggest alternatives (shared resources)
