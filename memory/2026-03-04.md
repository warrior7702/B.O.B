### API Path (for DOORY Phase 3)
1. GET /calendar/v2/resource_bookings?where[starts_at][gte]=...&include=resource → find Building Access (resource ID: 724356)
2. GET /calendar/v2/event_resource_requests?where[event_id]={id}&include=resource → find BA request IDs
3. GET /calendar/v2/event_resource_requests/{req_id}/answers → get the 4 question answers

### The 4 Questions (Building Access resource form)
1. "What type of access do you need?" — dropdown: Doors Unlocked | Badge Access | Door Code (multi-select)
2. "What entrance will your guest enter?" — (text/dropdown)
3. "Do you want a specific badge or code?" — (optional)
4. "Other details" — (optional text)

- `event_resource_requests` endpoint (not `resource_requests`) is the right one

- Building Access resource ID: 724356 (FBCA org)
